Frequency Domain (AC) Circuit Analysis

AC analysis for simple electronic circuits

Uses HTML5 Web Workers, tested in : WebKit,

Supports two formats of circuit netlist: a simple format described here (a simplified SPICE), and real SPICE format but with limitations (without component optional values and for now with numerical terminals).
